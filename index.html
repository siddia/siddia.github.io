Automated Car Parking

SCADA lab Project

This project demonstrates an automated car parking system. This will be installed at a existing parking lot with using 30% of space to yield same amount of parking spots. This system runs on one major controller that keeps track of all sensors, based on input the controller updates the HMI. There is a secondary motor driver that moves the stepper motors for the belt drive that carries the vehicles to each level. The system demonstrates SCADA methods learnt in the lab and combine them with HMI interface for user.

Mechanical Oscilloscope

Oscilloscope displays waves using a laser pointer

Oscilloscope is a laboratory instrument commonly used to display and analyze the waveform of electronic signals. In effect, the device draws a graph of the instantaneous signal voltage as a function of time. I made a Mechincal oscilloscope, that uses vibration of a speaker combined with horizontal rotation to display a sine wave using a laser pointer.

Random number Dice

Generates random number form 1 to 6, like a dice

Circuit generates a random number form 1 to 6. Idea is to make a electronic dice roll. The leds are attached to 4075 IC (Combinatorial Logic Chip) and 555 is used as a timer IC. The 555 sends pulse to 4075, then using combination of logic gates the leds turn on.

  • Category: Class Project

Robot Arm

Build a 4 axis robot arm with servo motors..

Based on industrial robot arms, This project was made using servos motors for 4 axis including base and claw motion. Each servo is connected to a arduino pwm pin, this gives the servo position.

The innivative aspect of this project was that I used bottle caps as gears. This worked as bottle caps have groves and each grove is equally spaced.

Project based on Denso industrial arm

A servomotor is a rotary actuator or linear actuator that allows for precise control of angular or linear position.

PLC

PLC Analog Control

Designed a PLC control solution to control and monitor the speed of an AC drive, using PowerFlex40 AC Drive. The PLC program can be used to change the desired speed for a motor. The program functions as follows: -Pressing the N.O. ‘UP’ pushbutton should increase the desired speed at a rate of 50RPM/sec -Pressing the N.O. ‘DOWN’ pushbutton should decrease the desired speed at a rate of 50RPM/sec.
